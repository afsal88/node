# ReST API starter using express, mongodb and apidoc

## Technology Stack
* Node 6.9.2
* Express 4.14.0
* MongoDB 3+

## Installation

* Clone the repo
* Remove .git directory
* Run npm start / yarn run start for starting the application
* Run npm apidoc / yarn run apidoc for genarate API documentation
